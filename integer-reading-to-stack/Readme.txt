Run the program from the jar file using the command line to avoid compilation issues (highly recommended)
1- Open terminal and navigate to the project root folder integer-reading-to-stack
2- Run the command in quotation " java -jar MyStackProgram.jar "

------------------------------------------------

How to Compile and run the Program from the Terminal

1- Open terminal and navigate to the project root folder integer-reading-to-stack
2- Run the following command with no quotation " javac -d . implementation/*.java "
3 - Run the following command with no quotation " java implementation.Main "

------------------------------------------------

How to create the documentation for the program 
1- Open terminal and navigate to the project root folder integer-reading-to-stack
2- Run the following command without quotation " javadoc -d doc implementation/*.java "
3- navigate to doc folder and open the index.html file in a browser
